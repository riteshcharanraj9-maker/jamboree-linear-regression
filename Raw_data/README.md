## 🔹 ABOUT

Jamboree is a renowned educational institution that has successfully assisted numerous students in gaining admission to top universities abroad.  

This project analyzes the factors influencing graduate admissions and builds machine learning regression models to predict the probability of admission for students.

---

## 🔹 Why this Case Study?

Solving this case study helps in understanding:

• Factors affecting graduate admission chances  
• Relationship between academic scores and admission probability  
• Building predictive machine learning models  
• Applying regression techniques to real-world datasets  

---

## 🤞 Work to be Done

As a data analyst / data scientist, the objective is to:

• Perform exploratory data analysis  
• Identify key factors influencing admission chances  
• Build regression models to predict admission probability  
• Provide insights and recommendations based on the analysis

## 📃 Dataset Features

| Feature | Description |
|--------|-------------|
| Serial No. | Unique identifier for each applicant |
| GRE Score | Graduate Record Examination score |
| TOEFL Score | TOEFL exam score |
| University Rating | Rating of university (1–5) |
| SOP | Statement of Purpose strength |
| LOR | Letter of Recommendation strength |
| CGPA | Undergraduate GPA |
| Research | Research experience (0 = No, 1 = Yes) |
| Chance of Admit | Probability of admission |

## ⭐ STAR Format Results

### 💡 Situation
Jamboree aims to improve student admission success by predicting the probability of admission to Ivy League colleges using historical applicant data.

### 💡 Task
Perform regression analysis to identify important predictors affecting admission chances and build a predictive model.

### 💡 Action

• Data cleaning and preprocessing  
• Handling missing values and duplicates  
• Exploratory Data Analysis (EDA)  
• Correlation analysis  
• Linear Regression model development  
• Ridge and ElasticNet regression implementation  
• Model evaluation using MAE, RMSE and R² score  

### 💡 Result

The analysis revealed that:

• CGPA, GRE Score, and TOEFL Score are the most influential predictors of admission probability  
• Research experience also improves admission chances  
• Regression models successfully explained around **82% of the variance in admission probability**

These insights can help students understand the key factors needed to improve their chances of admission.
